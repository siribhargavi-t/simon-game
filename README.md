# simon-game

## Description
Simon Game is a memory-based game built using HTML, CSS, and JavaScript. 
The game generates a sequence of colors and the player must repeat the same sequence correctly. 
Each level adds a new color to the sequence, making the game more challenging.

## Technologies Used
- HTML
- CSS
- JavaScript
- DOM Manipulation

- ## How to Play
1. Press any key to start the game.
2. The game will flash a color.
3. Click the same color.
4. The sequence increases every level.
5. If you click the wrong color, the game ends.
